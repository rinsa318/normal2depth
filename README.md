# normal2depth
Estimate depth from surface normal

  <img src="./figure/figure1.png" width=100%>
</div>

## Environment
Ubuntu 18.04  
Python3.8



## Dependency

+ OpenCV3
+ numpy
+ sys
+ os
+ scipy
+ argparse





## Usage



```
python main.py

optional arguments:
  -h, --help     show this help message and exit
  -i , --input   path to normal map image(OpenGL style)
  -m , --mask    path to mask image(optional)
```
